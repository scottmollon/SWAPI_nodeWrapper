(static) getAllResourcesByType(resource, page) → {json}

Returns a paged list of all items of a specific resource type. The resource type must be one of the types the API supports. If no page is provided when called the first page of results is returned.
Parameters:
Name Type Description
resource string The type of resource for which to retrieve items such as people, planets, vehicles.
page int The page of results to retrieve.
Source:
  • SW_API.js, line 121
Returns:
- A promise which will resolve a json response with a results list and information about number of and current page.
Type
json
Examples
//Get first page of results
var SWAPI = require('SW_API');
SWAPI.getAllResourcesByType('people').then(function(pagedResult){
  //do something with pagedResults of people
});
//Request a page of results
var SWAPI = require('SW_API');
SWAPI.getAllResourcesByType('people', 2).then(function(pagedResult){
  //do something with pagedResults of people
});
//Response
{
 'totalPages': 3,
 'currentPage': 2,
 'count': 26,
 'next': 'http://...../?page=3',
 'previous': 'http://...../?page=1',
 'results': []
}
